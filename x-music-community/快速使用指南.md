# 快速使用指南

## 立即开始

### 1. 清除缓存（重要！）
```
Windows: Ctrl + Shift + Delete
Mac: Cmd + Shift + Delete
```
选择"缓存的图片和文件"，点击"清除数据"

### 2. 硬刷新页面
```
Windows: Ctrl + F5
Mac: Cmd + Shift + R
```

### 3. 开始使用

## 核心功能

### 发布帖子
1. 点击"发布"按钮
2. 输入内容
3. 可选：上传图片/音频/视频
4. 点击"发布"

### 评论
1. 点击帖子下方的💬图标
2. 输入评论内容
3. 点击发送按钮✈️
4. 评论立即显示

### 转发
1. 点击帖子下方的🔄图标
2. 输入转发评论（可选）
3. 点击"转发"按钮
4. 等待提示"转发成功！刷新页面查看"
5. 点击右上角刷新按钮🔄或等待自动刷新

### 点赞
1. 点击帖子下方的❤️图标
2. 图标变红色
3. 点赞数 +1
4. 再次点击取消点赞

### 关注
1. 点击用户旁边的"关注"按钮
2. 按钮变为"已关注"
3. 你的关注数 +1
4. 对方的粉丝数 +1

### 刷新首页
1. 点击右上角的🔄按钮
2. 等待刷新完成
3. 新帖子出现在顶部

## 常见问题快速解决

### 页面显示异常
```
1. Ctrl + Shift + Delete 清除缓存
2. Ctrl + F5 硬刷新
3. 如果还不行，使用无痕模式
```

### 转发后不显示
```
1. 等待5-10秒（自动刷新）
2. 点击右上角刷新按钮
3. 或刷新浏览器页面
```

### 评论不显示
```
1. 点击💬图标展开评论区
2. 如果空白，刷新页面
3. 检查浏览器控制台错误（F12）
```

### 功能失败
```
1. 确保已登录
2. 检查网络连接
3. 刷新页面重试
4. 查看控制台错误信息
```

## 提示和技巧

### 1. 使用刷新按钮
- 转发后点击刷新按钮查看新帖子
- 比刷新整个页面更快

### 2. 查看评论
- 点击💬图标展开/收起评论区
- 评论按时间倒序显示

### 3. 转发格式
- 可以添加自己的评论
- 会自动引用原作者和内容
- 保留原帖的媒体内容

### 4. 避免页面崩溃
- 定期清除浏览器缓存
- 使用硬刷新而不是普通刷新
- 遇到问题先清除缓存

### 5. 检查错误
- 按 F12 打开控制台
- 查看红色错误信息
- 截图发送以便诊断

## 性能优化建议

### 开发时
- 使用无痕模式
- 禁用缓存（开发者工具 > Network > Disable cache）
- 定期清除缓存

### 使用时
- 关闭不用的标签页
- 定期刷新页面
- 避免同时打开多个实例
